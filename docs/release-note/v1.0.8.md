# v1.0.8 发行说明

## 功能增强

### 移动端体验优化

- **DNS 记录卡片布局**：移动端 DNS 记录列表从表格改为卡片样式，避免横向滚动，提升小屏设备的浏览体验
- **响应式布局状态保持**：修复屏幕宽度在断点切换时页面状态丢失的问题，现在调整窗口大小不会触发数据重新加载

### 更新体验优化

- **更新对话框**：检测到新版本时，弹出对话框显示版本号和发行说明，让用户了解更新内容后再决定是否更新

## 构建改进

- 支持手动触发 GitHub Actions 构建，可选择构建桌面端、Android 或全部平台
- 优化 Android APK 签名配置，支持通过环境变量传递签名信息

## 升级说明

从 v1.0.7 升级无需额外操作，直接安装新版本即可。
