# v1.3.2 发行说明

## 概述

v1.3.2 是一个针对阿里云 DNS Provider 的 bug 修复和性能优化版本。

## Bug 修复

### 阿里云 domain_id 不一致问题

修复 `get_domain` 和 `list_domains` 返回的 `id` 字段不一致的问题：

- **问题**：`list_domains` 返回域名名称作为 `id`，但 `get_domain` 返回真正的 `domain_id`
- **影响**：可能导致后续 API 调用使用错误的标识符
- **修复**：统一使用域名名称作为 `id`，与阿里云 API 的实际参数要求保持一致

## 性能优化

### 移除阿里云多余的 API 调用

优化 `list_records` 和 `create_record` 方法，移除不必要的 `get_domain` 调用：

- **优化前**：每次操作都先调用 `get_domain` 获取域名信息，再提取域名名称
- **优化后**：直接使用传入的 `domain_id`（它本身就是域名名称）
- **收益**：每次 `list_records` 和 `create_record` 操作减少一次 API 调用

## 技术细节

阿里云与其他 DNS Provider 的 ID 处理方式对比：

| Provider | ID 类型 | API 参数 |
|----------|--------|---------|
| Cloudflare | Zone ID（UUID） | Zone ID |
| Aliyun | 域名名称 | 域名名称 |
| DNSPod | 数字 ID | 域名名称 |
| Huaweicloud | Zone ID（UUID） | Zone ID |

阿里云是唯一一个使用域名名称作为标识符的 Provider，因此不需要额外的 ID 转换。

## 升级说明

从 v1.3.1 升级无需额外操作，直接安装新版本即可。
